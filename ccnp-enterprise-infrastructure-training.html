Course syllabus

1: Architecture

  • Explain enterprise network design (2-tier, 3-tier, fabric, cloud)
  • High availability: redundancy, FHRP, SSO
  • Wireless design principles: deployment models, location services, client density
  • SD-WAN: control/data plane elements, benefits, limitations
  • SD-Access: control/data plane elements, traditional campus interop

3: Infrastructure

  • Layer 2: Trunking, EtherChannel, Spanning Tree, root/BPDU guard
  • Layer 3: EIGRP vs OSPF, OSPFv2/v3 config, eBGP, PBR
  • Wireless: RF basics, AP modes, WLC discovery, L2/L3 roaming, WLAN troubleshooting
  • IP Services: NTP, NAT/PAT, HSRP/VRRP, multicast (RPF, PIM, IGMP)

4: Network Assurance

  • Device access control: lines, AAA
  • Infrastructure security: ACLs, CoPP
  • REST API security
  • Wireless security: 802.1X, WebAuth, PSK, EAPOL
  • Network security design: threat defense, endpoint, NGFW, TrustSec, MACsec, 802.1X, MAB

5: Security

  • Debug tools: traceroute, ping, SNMP, syslog
  • Flexible NetFlow, SPAN/RSPAN/ERSPAN
  • IPSLA, Cisco DNA Center workflows
  • NETCONF and RESTCONF

6: Automation

  • Python components and scripts
  • Construct JSON files
  • Data modeling: YANG
  • Cisco DNA Center and vManage APIs
  • REST API responses and payloads
