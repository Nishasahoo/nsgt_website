Customer Registration process and guidelines

Customer Registration process and guidelines

Step-by-Step Registration and Enrollment Process:

Create an Account

  • Visit the registration page at https://nitizsharma.com/dashboard.
  • Register using the same email address provided in the communication (e.g., email dropped in the WhatsApp conversation).

Select Your Course:

  • You will be given a course link that you're interested in.
  • Use the given course link to start the enrollment procedure.

Add Course to Cart:

  • Click on the course link to start the enrollment process.
  • Add the selected course to your cart.

Apply Coupon Code:

  • During the checkout process, enter the provided coupon code.
  • Ensure all billing details are correctly filled out.

Verify Cart Value:

  • After applying the coupon code, the cart value should adjust to zero.

Place the Order:

  • Complete the order placement process by clicking on the “Place Order” button.

Confirmation:

  • Once the order is placed, post a confirmation message in the designated communication group (e.g., WhatsApp conversation) indicating that the process is complete.

Approval:

  • An admin will review and approve your enrollment from their end.
